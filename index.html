INotebook

INotebook is a cloud based react application that stores an user's notes in the cloud. INotebook uses its own backend to store and update the notes of a particular user. User is required to type his password to access his notes.

NewsMonkey

NewsMonkey is a react application which fetches news from an API and displays the news in an organised way. The news can be sorted by the type. One can use this application to read news daily.

Fireapp

Fireapp is a basic android application which uses firebase's realtime database to store and update the data in it. Sensitive data like passwords are hidden and cannot be accesed by anyone else except the user. This application also has a splashscreen with the firebase logo.

TextUtils

TextUtils is also a react application which is a free character counter tool that provides instant character count & word count statistics for a given text. Thus it is suitable for writing text with word or character limit.
